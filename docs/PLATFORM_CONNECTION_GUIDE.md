# 平台连接优化 - 用户指南

## 🎯 改进概述

我们将平台连接流程从繁琐的手动配置优化为**三种简单方式**：

### 1. ✨ OAuth官方授权（推荐）
**适用平台：** X (Twitter)、YouTube、Meta等国际平台

**优点：**
- ✅ 最安全、最官方的授权方式
- ✅ 一键点击即可完成授权
- ✅ 自动刷新token，长期有效

**使用方法：**
1. 点击平台的"快速连接"按钮
2. 选择"简单模式"
3. 点击"使用官方授权登录"
4. 在弹出窗口中登录并授权
5. 完成！

### 2. 📱 扫码登录
**适用平台：** 微博、视频号等

**优点：**
- ✅ 无需输入密码
- ✅ 手机扫码即可授权
- ✅ 安全便捷

**使用方法：**
1. 点击平台的"快速连接"按钮
2. 选择"简单模式"
3. 点击"扫码登录"
4. 使用手机APP扫描二维码
5. 在手机上确认授权
6. 完成！

### 3. 🎁 一键导入登录状态（推荐）
**适用平台：** B站、小红书、抖音等

**优点：**
- ✅ 最快捷的方式
- ✅ 无需记住复杂的Cookie名称
- ✅ 自动检测登录状态

**使用方法：**
1. **首次使用：安装浏览器插件**
   - 访问插件目录：`/browser-extension`
   - 按照README说明安装插件
   - 只需安装一次！

2. **连接平台：**
   - 在浏览器中登录要连接的平台（如B站）
   - 打开AI媒体助手的平台管理页面
   - 点击"快速连接" → "简单模式" → "一键导入登录状态"
   - 插件自动导入Cookie，瞬间完成！

### 4. 🔧 高级模式（专业用户）
如果您了解API配置，仍可使用手动输入方式：
1. 点击"高级模式"
2. 手动输入认证信息
3. 参考平台提供的教程获取Cookie

## 📦 浏览器插件安装指南

### Chrome/Edge用户

1. 打开扩展管理页面
   - Chrome: 访问 `chrome://extensions/`
   - Edge: 访问 `edge://extensions/`

2. 开启"开发者模式"（右上角开关）

3. 点击"加载已解压的扩展程序"

4. 选择项目中的 `browser-extension` 文件夹

5. 完成！插件图标会出现在浏览器工具栏

### 使用提示

- 💡 插件只在您点击"导入登录状态"时才会读取Cookie
- 🔒 数据仅发送到本地应用，绝不上传到其他服务器
- ⚡ 一次安装，永久使用
- 🎨 点击插件图标可以查看各平台登录状态

## ❓ 常见问题

**Q: 浏览器插件安全吗？**
A: 完全安全！插件代码开源，仅在您明确授权时读取Cookie，且数据仅发送到本地应用。

**Q: 为什么还有"高级模式"？**
A: 为专业用户和特殊场景保留。如果您偏好手动配置或插件无法使用，仍可选择高级模式。

**Q: OAuth授权和Cookie导入有什么区别？**
A: OAuth是官方标准授权方式，更安全且会自动刷新；Cookie导入更快捷但需要定期重新导入。

**Q: 可以同时连接多个账号吗？**
A: 目前每个平台支持一个账号。未来会支持多账号管理。
