## 一、快速安装

### 1.1 桌面安装包（推荐）

| 平台 | 文件 | 安装方式 |
|------|------|----------|
| **macOS** | `AI-Media-Agent-1.0.36.dmg` | 双击挂载 → 拖入 Applications |
| **Windows** | `AI-Media-Agent-1.0.36.zip` | 解压 → 运行 `AI Media Agent.exe` |

安装完成后首次启动：
1. 系统可能会提示"无法验证开发者" — 点击「仍要打开」
2. 自动检查并安装 Playwright 浏览器依赖（约 2-5 分钟）
3. 浏览器窗口自动打开 http://localhost:3000

### 1.2 源码安装（开发者）

```bash
git clone https://github.com/phuhao00/ai-media-agent.git
cd ai-media-agent
./start_local.sh
```

## 二、首次配置

### 2.1 配置 AI 模型 API Key

启动后，在侧边栏点击 **⚙️ 设置 → 模型设置**。

至少需要配置一个模型供应商：

| 供应商 | 用途 | 获取地址 |
|--------|------|----------|
| **智谱 AI** | GLM-4 / CogView / CogVideoX | [open.bigmodel.cn](https://open.bigmodel.cn) |
| **OpenRouter** | GPT-4o / Claude / Gemini 等 100+ 模型 | [openrouter.ai](https://openrouter.ai) |
| **即梦 AI** | 图片/视频生成 | [jimeng.jianying.com](https://jimeng.jianying.com) |
| **豆包** | SeaDance 视频生成 | [console.volces.com](https://console.volces.com) |

> 💡 **建议**：至少配置智谱 + OpenRouter，可覆盖 90% 场景。

### 2.2 配置平台连接器

在侧边栏 **⚙️ 设置 → 平台管理** 中配置：

| 平台 | 认证方式 | 操作步骤 |
|------|----------|----------|
| 小红书 | Cookie | 浏览器 F12 → Application → Cookies → 复制 `a1` 字段 |
| 抖音 | Cookie | 同上 |
| B站 | Cookie | 复制 `SESSDATA` 字段 |
| YouTube | OAuth2 | 点击「授权」→ 跳转 Google 登录 → 自动回调 |
| Twitter/X | API Token | 在 [developer.twitter.com](https://developer.twitter.com) 创建 App |

配置后点击「测试连接」验证是否生效。

## 三、核心功能使用

### 3.1 AI 对话与创作

**入口**：侧边栏 🏠 **首页** 或 **工作台**

**操作步骤**：
1. 在底部输入框输入需求，如："写一篇小红书风格的种草文案，关于一款降噪耳机"
2. 在顶部模型选择器切换模型（GLM-4 / GPT-4o / Claude 等）
3. 按 **Enter** 发送，AI 实时流式输出
4. 点击消息右上角 📋 复制内容

**进阶技巧**：
- 输入 `@` 可调用特定技能（如 `@xiaohongshu-operator`）
- 拖拽图片到输入框可进行多模态对话
- 使用 `/settings/context` 查看知识图谱和记忆管理

### 3.2 AI 图片生成

**入口**：侧边栏 🎨 **媒体生成 → 图片**

**操作步骤**：
1. 选择后端模型（CogView / 即梦 / Gemini Image）
2. 输入提示词，如："一只戴着墨镜的猫咪，赛博朋克风格，霓虹灯背景"
3. 点击「生成」等待 10-30 秒
4. 右键图片 → 保存或发布到平台

**图生图**：
- 上传参考图片 → 输入修改指令 → 生成变体

### 3.3 AI 视频生成

**入口**：侧边栏 🎬 **媒体生成 → 视频**

**支持类型**：
| 类型 | 入口 | 说明 |
|------|------|------|
| 文生视频 | 视频页 | 输入文字描述生成视频 |
| 图生视频 | 视频页 | 上传图片生成动态视频 |
| 故事板 | 故事板 | 分镜脚本 → AI 配图 → 自动拼接 |
| 长视频 | 长视频 | 30-120 秒长视频生成 |
| 自动混剪 | 视频 → AI 混剪 | 上传素材，AI 自动剪辑配字幕 |

### 3.4 多平台发布

**入口**：侧边栏 📢 **平台管理**

**一键发布流程**：
1. 在 AI 对话或创作中心生成内容
2. 点击内容卡片上的「发布」按钮
3. 选择目标平台（可多选）
4. 预览并确认
5. 点击「立即发布」

**批量发布**：
- 在爆款流水线中配置「发布」节点，自动推送到多个平台

### 3.5 定时发布

**入口**：侧边栏 ⏰ **定时发布**

**创建定时任务**：
1. 点击「新建任务」
2. 选择调度方式：
   - **Cron 表达式**：`0 9 * * *`（每天早9点）
   - **固定间隔**：每 6 小时
3. 选择内容类型（图片 / 视频 / 软文）
4. 选择目标平台
5. 保存任务

**快捷预设**：
- 每天早9点发图 → 自动生成 AI 图片发布
- 每6小时发图 → 持续热点追踪
- 每天发视频 → 自动化视频运营
- 每周一软文 → 品牌内容更新

### 3.6 爆款流水线

**入口**：侧边栏 🏭 **爆款流水线**

**一键式内容生产**：
1. 点击「新建流水线」
2. 选择模板：
   - 热点追踪 → 文案 → 配图 → 发布
   - 选题 → 脚本 → 视频 → 发布
   - 自定义工作流
3. 配置每个节点的参数
4. 点击「运行」或「保存为定时任务」

### 3.7 RAG 知识库

**入口**：侧边栏 📚 **知识库**

**上传文档**：
1. 点击「上传文档」或拖拽文件到页面
2. 支持格式：PDF、Word、Markdown、TXT、CSV、Excel
3. 系统自动解析并向量化
4. 在 AI 对话中，AI 会自动检索知识库内容作为上下文

**管理知识库**：
- 查看已上传文档列表
- 点击文档进入详情页，查看分块和 FAQ
- 一键优化文档（自动补全缺失内容）
- 导入 FAQ 到分类中

### 3.8 游戏热点

**入口**：侧边栏 🔥 **游戏热点**

**实时数据**：
- **Steam**：畅销榜、新品榜、特惠游戏
- **Epic**：本周免费游戏
- **TapTap**：热门游戏、新游预约

**自动化操作**：
1. 点击「刷新」获取最新数据
2. 点击某款游戏 → AI 自动生成选题建议
3. 一键生成图文/视频内容

### 3.9 Labs 专业助手

**入口**：侧边栏 🧪 **Labs**

| 助手 | 用途 | 场景 |
|------|------|------|
| **产品经理** | 需求分析、PRD 撰写、竞品分析 | 产品策划 |
| **法务顾问** | 合同审查、合规建议、法律检索 | 商务合作 |
| **采购助手** | 供应商比价、采购流程、合同管理 | 采购管理 |
| **桌面操作** | 原生 GUI 自动化、文件操作 | 系统维护 |
| **商务合作** | 合作方案、商务谈判、资源对接 | 商务拓展 |
| **广告投放** | 广告策略、素材优化、投放分析 | 营销推广 |

**使用方式**：
1. 进入 Labs 选择对应助手
2. 描述你的需求或上传相关文件
3. AI 助手按专业流程输出结果

## 四、高级功能

### 4.1 Computer Use 浏览器自动化

**入口**：侧边栏 🌐 **Computer Use**

**功能**：让 AI 像真人一样操作浏览器，完成复杂网页任务。

**使用场景**：
- 自动填写表单
- 批量采集网页数据
- 自动化测试网站
- 在无法 API 化的平台上操作

**操作步骤**：
1. 输入任务描述，如："打开小红书创作者中心，查看最近7天的数据报表"
2. AI 自动规划操作步骤
3. 每一步都会显示截图和说明
4. 高危操作（如发布、删除）需人工审批

### 4.2 AI 伙伴（数字人）

**入口**：侧边栏 🧑‍💻 **AI 伙伴**

**功能**：可交互的数字人 companion，支持语音对话与办公场景。

**操作**：
- 点击麦克风图标进行语音对话
- 切换不同角色（助理/顾问/伙伴）
- 设置办公场景背景

### 4.3 OpenClaw 分布式 Agent

**入口**：侧边栏 🦞 **OpenClaw**

**功能**：多节点分布式 Agent 协作网络，支持跨机器协作。

**使用场景**：
- 多账号矩阵运营
- 分布式内容采集
- 团队协作任务分配

### 4.4 My Context 知识图谱

**入口**：侧边栏 ⚙️ **设置 → My Context**

**功能**：可视化展示知识关联，辅助内容创作。

**操作**：
- **知识图谱**：查看后端拼装的实体关系图
- **Memory 记忆**：检索、筛选与管理上下文记忆条目
- **CodeGraph**：代码调用关系图（开发场景）

## 五、故障排除

### 5.1 常见问题

| 问题 | 原因 | 解决方法 |
|------|------|----------|
| 启动后白屏 | Playwright 未安装 | 等待自动安装完成，或手动运行 `playwright install` |
| 平台发布失败 | Cookie 过期 | 重新抓取平台 Cookie 并更新 |
| 图片生成慢 | 网络或模型队列 | 切换模型后端，或检查 API Key 余额 |
| 视频生成失败 | 参数超出范围 | 检查视频时长是否在 30-120 秒范围内 |
| 知识库搜索无结果 | 文档未索引完成 | 等待文档解析完成（大文件可能需要几分钟） |
| AI 对话无响应 | SSE 连接中断 | 刷新页面或检查后端服务是否运行 |
| 定时任务未执行 | 调度器未启动 | 检查 `backend/services/scheduler.py` 是否运行 |

### 5.2 日志查看

```bash
# 实时查看后端日志
tail -f logs/agent.log

# 查看 Electron 日志（桌面版）
# macOS: ~/Library/Logs/ai-media-agent/
# Windows: %APPDATA%\ai-media-agent\logs\
```

### 5.3 重置配置

```bash
# 重置所有本地数据（谨慎操作）
rm -rf storage/ outputs/ logs/
```
