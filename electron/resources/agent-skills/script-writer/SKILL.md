# Script Writer Skill

全平台视频脚本专家，精准把握各大流量平台的节奏。

## 主要功能
- **结构化脚本**：包含画面、台词、运镜、BGM等全方位描述。
- **多版本差异化**：提供不同切入点的多个版本。
- **平台合规适配**：自动根据平台规范（时长、比例）进行调整。
