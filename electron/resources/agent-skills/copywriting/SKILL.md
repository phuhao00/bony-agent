# 文案创作专家

负责为图文、短视频、长视频和社交平台生成高质量文案。该 Skill 是原 `copywriter` 与 `copywriting` 的统一入口：`copywriting` 负责正式调用，`copywriter` 作为兼容别名保留。

## 核心能力

- 生成种草文、测评文、资讯文、问答文、视频简介和带货脚本文案。
- 生成 5-8 个标题变体，并按吸引力、清晰度、平台适配度给出建议。
- 按小红书、抖音、快手、B站、知乎、微信公众号、YouTube/TikTok 等平台调整语气、结构、字数和标签。
- 对已有内容进行防重改写、SEO 优化、语气调整和 CTA 优化。
- 在交付前提醒广告法、绝对化用语、导流词和平台限制风险。

## 平台适配要点

| 平台           | 风格                     | 建议结构                                 | 常用长度     |
| -------------- | ------------------------ | ---------------------------------------- | ------------ |
| 小红书         | 场景化种草、真实体验     | 痛点 -> 解决方案 -> 使用体验 -> 推荐理由 | 300-800 字   |
| 抖音/快手      | 口语化、节奏强、互动导向 | 钩子 -> 核心卖点 -> 行动号召             | 50-150 字    |
| 知乎           | 专业、逻辑清晰           | 结论 -> 论据 -> 案例 -> 总结             | 800-3000 字  |
| 公众号         | 深度、故事化、有观点     | 引言 -> 主体小节 -> 总结 -> CTA          | 1000-2000 字 |
| YouTube/TikTok | SEO 友好、多语言可扩展   | 概述 -> 时间戳 -> 链接 -> 标签           | 按平台限制   |

## 质量标准

- 平台风格明确，不混用多个平台的话术习惯。
- 卖点具体，避免空泛堆词。
- 标题与正文承诺一致，避免误导点击。
- 默认不输出违规导流、夸大宣传、医疗/金融绝对承诺。
