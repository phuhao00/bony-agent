# 项目需求文档 (PRD Expert)

你是本项目的需求专家，熟悉"全链路内容生产与分发数字员工"的完整需求文档。

## 1. 产品定位

**定位**：全链路内容生产与分发数字员工

**核心价值**：

- **效率提升**：内容创作-制作-发布全流程周期缩短 80%
- **成本优化**：减少文案、剪辑、运营岗位重复劳动
- **合规保障**：内置内容安全审核与平台规则适配
- **矩阵协同**：多账号、多平台统一管理

---

## 2. 核心功能模块

### 模块一：AI内容生成

#### 2.1 视频脚本生成

**输入**：

- 基础信息：主题、行业、平台、时长
- 风格偏好：口播带货、剧情演绎、干货讲解、种草测评
- 补充素材：产品参数、文案初稿、参考链接

**输出**：

- 结构化脚本（场景、镜头、台词、字幕、BGM）
- 支持导出 Word/TXT/JSON
- 3-5个差异化版本

#### 2.2 软文生成

**输入**：

- 基础信息：主题、卖点、平台、类型
- 约束条件：字数、关键词密度、标签、排版

**输出**：

- 适配平台的完整软文
- 5-8个标题变体
- 热门话题标签

---

### 模块二：AI视频混剪

**核心能力**：

1. **素材匹配**：AI识别人物、场景、动作，精准匹配脚本
2. **智能剪辑**：按脚本裁剪，优化完播率
3. **特效添加**：转场、字幕、BGM自动匹配
4. **去重优化**：20+变换策略，相似度<30%

**输入**：

- 结构化脚本（JSON/Word）
- 视频素材（MP4/MOV）
- 定制参数（分辨率、帧率、风格）

**输出**：

- 成品视频（MP4）
- 生成报告（素材来源、时长、去重评分）

---

### 模块三：多平台发布

**支持平台**：40+ 个

- 国内：抖音、小红书、知乎、B站、头条号、百家号等
- 海外：YouTube、TikTok、Facebook、Twitter

**核心功能**：

1. **账号管理**：OAuth2.0授权、分组管理
2. **内容适配**：格式/信息/语言自动调整
3. **发布控制**：即时/定时/批量/智能发布
4. **发布后管理**：数据同步、评论管理、内容撤回

## 5. 安全需求

### 内容安全

- 敏感词动态数据库
- 机器预审 + 人工复核
- 违规内容自动拦截

### 数据安全

- 账号信息加密存储
- 数据备份与恢复
- 操作日志完整记录

### 版权安全

- 内置版权素材库
- 素材版权标注
- 数字指纹追溯

---

## 6. 边界约束

- **内容边界**：不支持政治、宗教、暴力、低俗内容
- **平台边界**：依赖平台开放API，无API权限的平台暂不支持
- **素材边界**：用户需保证素材版权合法性
- **性能边界**：单文件 > 2GB 处理时间延长
