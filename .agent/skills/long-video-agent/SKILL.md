# 长视频工坊专家

负责把自然语言脚本转化为较长的一整条视频。该 Skill 是 `media_agent` 的长视频垂直入口：当用户需要多镜头、可分镜、连续叙事的成片时使用本 Skill；几秒钟的短视频应使用普通视频生成。

## 核心能力

- **多分镜规划**：将脚本拆分为可连续拍摄/生成的镜头序列。
- **分段并行生成**：基于通义 Wan 等模型并行生成各分镜片段。
- **成片拼接**：将分段视频按时间线拼接为完整长视频。
- **叙事节奏控制**：根据脚本调整镜头时长、转场与情绪曲线。

## 典型工作流

1. 接收用户脚本或主题，确认视频长度、风格与画幅。
2. 生成分镜表（镜头编号、画面描述、时长、旁白/字幕）。
3. 调用 `produce_long_video` 分段并行生成视频片段。
4. 拼接成片，输出最终视频路径与分镜说明。

## 输入/输出约定

- 默认 `duration_sec = 30`，风格 `cinematic`；用户明确表达长度时才传入自定义秒数（范围 30–120）。
- 输出包含：成片路径、分镜表、各段生成状态。

## 质量标准

- 分镜之间叙事连贯，角色与场景一致。
- 成片音画同步，字幕/配音（如有）对齐。
- 临时文件只使用 `storage/temp/`，不使用系统 `/tmp`。
