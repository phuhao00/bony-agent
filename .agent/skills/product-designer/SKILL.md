## 二、用户流程设计

### 完整业务流程

```
┌─────────┐    ┌─────────┐    ┌─────────┐    ┌─────────┐    ┌─────────┐
│  需求   │ → │ 内容    │ → │ 视频    │ → │ 内容    │ → │ 多平台  │
│  输入   │    │ 生成    │    │ 混剪    │    │ 审核    │    │ 发布    │
└─────────┘    └─────────┘    └─────────┘    └─────────┘    └─────────┘
     │              │              │              │              │
     ▼              ▼              ▼              ▼              ▼
  主题/行业     脚本+软文      成品视频      安全检查      自动适配
  平台/素材     多版本生成    去重优化      规则适配      定时发布
                                                              ↓
                                                         ┌─────────┐
                                                         │ 数据    │
                                                         │ 分析    │
                                                         └─────────┘
```

### 关键用户故事

#### Story 1: 快速生成短视频内容

```
作为 MCN运营人员
我希望 输入产品信息后自动生成脚本和视频
以便 快速产出多个差异化内容用于矩阵分发
```

#### Story 2: 多平台一键发布

```
作为 品牌营销经理
我希望 一次操作发布到多个平台
以便 节省逐个平台发布的时间，实现矩阵运营
```

#### Story 3: 内容合规保障

```
作为 内容审核人员
我希望 系统自动检测违规内容
以便 避免发布后被平台限流或下架
```

## 四、设计规范

### 颜色系统

```css
/* 主色 */
--primary: #6366f1; /* 品牌紫 */
--primary-dark: #4f46e5;

/* 功能色 */
--success: #22c55e; /* 成功/通过 */
--warning: #f59e0b; /* 警告/待处理 */
--error: #ef4444; /* 错误/失败 */
--info: #3b82f6; /* 信息/提示 */

/* 中性色 */
--gray-50: #f9fafb;
--gray-100: #f3f4f6;
--gray-900: #111827;
```

### 组件规范

- 按钮：圆角 8px，主按钮使用 primary 色
- 卡片：圆角 12px，阴影 shadow-md
- 输入框：圆角 8px，focus 时显示 primary 边框
- 表格：斑马纹，hover 高亮

### 响应式断点

```css
sm: 640px   /* 手机横屏 */
md: 768px   /* 平板竖屏 */
lg: 1024px  /* 平板横屏 */
xl: 1280px  /* 桌面 */
2xl: 1536px /* 大屏 */
```
