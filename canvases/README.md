# Cursor Canvas 源码目录

本目录存放 **纳入 Git 的 Canvas 源码**。Cursor IDE 实际渲染的文件在：

```text
~/.cursor/projects/<workspace>/canvases/
```

修改后请同步，否则 IDE 中可能仍显示旧版。完整说明见 **[`docs/CANVAS_OVERVIEW.md`](../docs/CANVAS_OVERVIEW.md)**。

| 文件 | 说明 |
|------|------|
| `ai-media-agent-overview.canvas.tsx` | 项目工作区速览（彩色图表 + 架构 SVG + API/路由速查） |
